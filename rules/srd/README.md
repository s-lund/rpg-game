# Vendored SRD reference (M2 subset)

Reference excerpts from **Pathfinder Player Core** (ORC License), curated for EMBERWATCH M2 character creation (level 1 Fighter and Rogue only).

Machine-readable tables live in [`m2-subset.json`](m2-subset.json). Core code imports that file; these markdown files are human-readable ground truth with section citations.

**Source:** [Archives of Nethys](https://2e.aonprd.com/) — official PF2e rules reference (ORC-licensed remaster content). Implementations must match the vendored subset, not memory.
