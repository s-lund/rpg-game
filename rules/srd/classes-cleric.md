# Cleric (level 1 excerpt)

**Source:** Pathfinder Player Core, Class — Cleric (ORC).

## Key ability

Wisdom.

## Hit Points

8 + Constitution modifier at 1st level.

## Skills (1st level)

Trained in a number of additional skills equal to **2 + Intelligence modifier**.

## Spells (M7 subset)

- **Heal** spell, 2-action version (30 feet, 1d8+8 to a willing ally) — see `spell-heal.md`
